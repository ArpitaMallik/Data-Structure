# Data Structures in C/C++

This repository contains manual implementations of fundamental data structures in C++. The objective is to build strong foundational understanding by implementing each structure from scratch rather than relying on STL abstractions.

## Implemented Structures

* Arrays
* Linked Lists (Singly, Doubly, Circular)
* Stack
* Queue (Linear, Circular, Deque)
* Binary Tree
* Binary Search Tree
* Graph (Adjacency List/Matrix, BFS, DFS)

Each implementation includes core operations, example usage, and time/space complexity considerations.

## Goals

* Strengthen problem-solving fundamentals
* Understand internal working mechanisms
* Analyze algorithmic complexity
* Build a solid base for interviews and advanced CS topics
